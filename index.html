Why BidVision?

Your bid sheet has over a thousand sequences. BidVision helps you find the ones that actually fit your life — faster and more flexibly than anything else out there.

📅

Build your schedule around your life

Mark the days that matter — a wedding, a trip, your kid's recital — and BidVision finds trips that work around them. Or prioritize the trips you like and see which days they fall on. It works either way.

🔍

More flexible than what you're using now

BidVision is built for how we actually bid — with more filters, more control, and more ways to search than the current tools offer. Updated, robust, and designed around real crew needs.

💎

Soft filters find hidden gems

Most tools use hard yes-or-no filters that hide everything else. BidVision lets you prefer, allow, or exclude — so trips that almost match still show up, scored and ranked. You might find a sequence you'd never have seen otherwise.

BidVision is a pre-bid research tool — use it anytime to explore your options before the bidding window opens.

Download BidVision for your platform.

BidVision is a desktop app — open this page on your computer to download.
 macOS (Apple Silicon) M1, M2, M3, M4 Macs — 2020 or newer  macOS (Intel) Pre-2020 Macs ⊞ Windows Windows 10 or later
⚠ macOS will block BidVision the first time. After unzipping, try to open the app. Then go to System Settings → Privacy & Security, scroll down to "BidVision was blocked," and click Open Anyway. You only need to do this once.

Click the Apple menu () in the top-left corner of your screen, then About This Mac.

  • If it says M1, M2, M3, or M4 → choose Apple Silicon
  • If it says Intel (or doesn't mention a chip) → choose Intel

Most Macs from 2020 onward are Apple Silicon. If yours is older than that, it's almost certainly Intel.

macOS

  1. Unzip the downloaded file
  2. Drag BidVision to your Applications folder
  3. Try to open it — macOS will block it the first time
  4. Open System Settings → Privacy & Security
  5. Scroll down — you'll see "BidVision was blocked." Click Open Anyway
  6. Click Open in the confirmation dialog
  7. After the first launch, it opens normally

This happens because BidVision isn't code-signed yet — it's safe to proceed. Code signing is planned for the public release.

Windows

  1. Run the downloaded BidVision-Windows-Setup.exe
  2. If SmartScreen appears, click More info then Run anyway
  3. BidVision installs and opens automatically
  4. Find it in your Start Menu from now on
